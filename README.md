## 1.0 名词介绍
在计算广告场景中，需要平衡和优化三个参与方——用户、广告主、平台的关键指标。

而预估点击率CTR（Click-through Rate）和转化率CVR（Conversion Rate）是其中非常重要的一环。

准确地预估CTR和CVR对于提高流量变现效率，提升广告主ROI（Return on Investment），保证用户体验等都有重要的指导作用。
